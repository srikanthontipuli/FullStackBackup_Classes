Multiple callback functions would create callback hell that leads to unmanageable code. Promises are used to handle asynchronous operations in JavaScript. The promise constructor takes only one argument which is a callback function The callback function takes two arguments, resolve and reject Perform operations inside the callback function and if everything went well then call resolve. If desired operations do not go well then call reject.
